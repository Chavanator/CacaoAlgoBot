# Changelog

## [1.0.0] - 2026-06-15

### Added
- Initial public release of Cacao_QuantBot
- Repository restructured from original `D:\iqrobot` into professional layout

### Changed
- **Sanitized sensitive credentials**: Removed hardcoded email/password from `config.txt`, `aula3.py`, `borradores/robot_git.py`, `borradores/aula2.py`. Replaced with environment variable references.
- **Restructured codebase**: Flattened 197 Python scripts into organized `src/` modules (brokers/, strategies/, ml_models/, risk/, utils/) and `scripts/` executables.
- **Consolidated redundant scripts**: Merged 6+ Linear Regression variants into `src/ml_models/linear_regressor.py`. Merged 4+ LSTM variants into `src/ml_models/lstm_predictor.py`.
- **Standardized naming**: Function names, variables, and comments translated from Portuguese/Spanish to English.

### Removed
- `__pycache__/` directories (126 .pyc files) - auto-generated bytecode
- `build/`, `dist/` - empty PyInstaller artifacts
- `df_test/` (7,692 CSV files, 428 MB) - large historical market data
- `borradores/` (31 experimental scripts) - unmaintained drafts
- `iqoptionapi-master/` - external fork (original `iqoptionapi/` retained)
- `prueba/`, `resultados/` partial exports
- `app.py`, `app2.py`, `app3.py`, `app_graph.py` - unrelated student project
- `12345.zip`, `PROJECT__AI-Binary-Options-Trading-Bot-main.zip` - external archives
- `notebooks/chromeremotedesktophost.msi` - installer binary
- Various temp/empty files (`correcviones.txt`, `Iniciando`, `Orden`, `predicted_price`)
- 29 generated PNG figures (reproducible from notebooks)
- `bot.py` (superseded by `bot17.py`)

### Security
- Added `.env.example` template for all credentials
- All secrets now loaded via `os.getenv()` from `.env` file
- Added `.gitignore` to prevent committing `.env`, `__pycache__`, credentials
